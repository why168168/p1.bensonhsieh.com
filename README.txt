Features:
---------
a) Images can be made to slide vertically or horizontally.
b) Options to add shrink/grow and fade effects.
c) Clicking the thumbnail images at the bottom will display that image in the current window.
d) Both pictures and text can be used in this featured content slider.
e) Manual sliding option is provided.
f) Fully customized and interactive image slider.
g) Simple and easy to configure Jquery script.
Downloads:
-----------
Please visit our site http://www.hscripts.comscripts/jquery/feature-content-slider.php and do the download


Extracting files:

a) Unzip feature-content.zip.
b) You will get  feature-image-slider/jquery.js,feature-image-slider/index.php,feature-image-slider/README.txt,feature-image-slider/style.css and images.

Embedding the code:

a) Now run the feature-image-slider/index.php in your browser.

b) Featured Image | Content Slider Script is now ready to work fast and efficient.

c) If you want to change the title and the description of the images, you can change the "title" and "alt" tags of the images in index.php file.

d) If you want to change the speed of the slider, duration of autoplaying and autoplaying option of the images, then you can change the below variables in index.php file.            

                               "speed","duration","autostart"
